Introductie

Doelstelling

Startinstructies

  • Volg de instructies!!!!!

Gitflow

Afbeeldingen toevoegen

Opdracht

Voor je begint met de opdracht, zorg dat je in de develop branch zit.

Voltooi de volgende feature

  • Herhaal de volgende stappen voor iedere hoofdstuk.
    • Zoek een passende afbeelding voor het hoofdstuk, voeg deze afbeelding toe in de map images en maak een commit. zoeken
    • Pas de src aan van de placeholder afbeelding van het introductie hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is en maak een commit.

Hete soep!

Voltooi de volgende hotfix

  • Voeg jouw naam toe aan de index.html
    Boven aan de pagina, in de header, kun je de tekst "Student-naam" weghalen en jouw eigen naam invoeren.

Een conflict oplossen

Winnaar: Chuck Norris

Chuck: Begin de volgende feature

  • Pas de hierboven aangegeven winnaar aan naar: Chuck Norris
  • Commit deze aanpassing, maar merge het nog niet

Superman: Voltooi de volgende feature

  • Pas de hierboven aangegeven winnaar aan naar: Superman
  • Commit deze aanpassing, maar merge het nog niet

Fight!

  • Merge Superman feature naar develop.
  • Controleer of superman de winnaar is.
  • Merge Chuck feature naar develop.
  • Conflict! 
    
                                <<<<<<< HEAD
                                <span id="winnaar">Superman</span>
                                =======
                                <span id="winnaar">Chuck</span>
                                >>>>>>> feature/chuck
  • Los het conflict zodanig op dat Chuck de winnaar is.

Release

Voltooi de volgende release

  • Pas de versie aan in de index.html
    Boven aan de index.html, zie je de laatse versie, jouw {versie} is 1 cijfer hoger.
  • Push de develop branch naar github
  • Push de master branch naar github
